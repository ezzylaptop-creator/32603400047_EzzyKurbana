Project: Sistem CRUD Data Mahasiswa dengan CodeIgniter 4.

File database:
- database/mahasiswa.sql

Langkah singkat menjalankan:
1. Import database/mahasiswa.sql ke MySQL/phpMyAdmin.
2. Sesuaikan .env jika username/password MySQL berbeda.
3. Jalankan: php spark serve
4. Buka: http://localhost:8080/crud
